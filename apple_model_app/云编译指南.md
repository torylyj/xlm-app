# 🌐 云编译配置指南

## 📦 简介

本项目支持使用**GitHub Actions**进行云编译，无需本地 macOS 环境即可编译 iOS 应用。

---

## 🚀 快速开始

### 步骤 1：创建 GitHub 仓库

```bash
# 1. 在 GitHub 创建新仓库
# 2. 上传项目文件

# 上传方式：
git clone https://github.com/yourusername/apple_model_app.git
cd apple_model_app
git add .
git commit -m "Initial commit"
git push origin main
```

### 步骤 2：添加编译脚本

编译脚本已包含在：
- `.github/workflows/build.yml`

### 步骤 3：触发编译

```bash
# 方法 1：在 GitHub 仓库提交新代码
# Actions 自动触发编译

# 方法 2：手动触发编译
cd apple_model_app/.github/workflows
gh run dispatch
```

## 📋 编译参数

| 参数 | 说明 | 默认值 |
|-----|------|-------|
| `scheme` | 编译方案名 | AppleModelApp |
| `configuration` | 编译配置 | Release |
| `destination` | 目标平台 | iphoneos |
| `archivePath` | 归档路径 | build/AppleModelApp.xcarchive |
| `clean` | 是否清理 | false |

## 🚀 编译命令

### 自动编译

```bash
# 使用 Python 脚本编译
python3 scripts/build_app.py \
  --scheme AppleModelApp \
  --configuration Release \
  --destination iphoneos \
  --workspace apple_model_app.xcworkspace
```

### 手动编译（Xcode）

```bash
# 使用 Xcode
xcodebuild \
  -workspace AppleModelApp.xcworkspace \
  -scheme AppleModelApp \
  -configuration Release \
  -sdk iphoneos \
  -archivePath build/Build.xcarchive \
  archive
```

## 🎯 自动化部署

### GitHub Actions 脚本

```yaml
# .github/workflows/build.yml
name: Build iOS App

on:
  push:
    branches: [ main ]
  pull_request:
    branches: [ main ]
  workflow_dispatch:

jobs:
  build:
    runs-on: macos-latest
    
    steps:
    - uses: actions/checkout@v4
    
    - name: Build app
      run: |
        xcodebuild \
          -workspace apple_model_app.xcworkspace \
          -scheme AppleModelApp \
          -configuration Release \
          -archivePath build/Build.xcarchive \
          archive
    
    - name: Upload artifacts
      uses: actions/upload-artifact@v4
      with:
        name: iOS-Binaries
        path: build/Build.xcarchive
    
    - name: Publish to App Store
      if: ${{ github.event_name == 'workflow_dispatch' }}
      run: |
        # Submit to App Store Connect
        # Review and approval required
```

## 🔐 安全性

### SSH 密钥配置

```bash
# 配置 SSH 密钥
ssh-keygen -t ed25519
cat ~/.ssh/id_ed25519.pub

# 添加到 GitHub
# https://github.com/settings/ssh
```

### 编译证书

```bash
# 生成证书
security create-keychain -p password default.keychain
security default-keychain -s default
security unlock-keychain -p password default.keychain

# 创建代码签名 ID
security create-certificate-asking-permission *.p12
security import *.p12 -f passcode
```
