## 🎯 简介

本项目使用**魔搭社区（ModelScope）**作为模型仓库，支持离线安装和使用。

## 🌐 下载方式

### 方式 1：使用 MLX 库下载

```bash
# 1. 安装 MLX
pip install mlx-lm

# 2. 使用国内镜像下载
python3 -c "import mlx_lm"

# 3. 下载模型
from mlx_lm.models import download_model
download_model("qwen2.5-7b-4bit", "Resources/Models")
```

### 方式 2：使用 HuggingFace

```bash
# 1. 安装 huggingface_hub
pip install huggingface_hub

# 2. 下载模型
python3 -c "
from huggingface_hub import hf_hub_download
model_path = hf_hub_download(
    repo_id='Qwen/Qwen2.5-7B-4bit',
    filename='model.safetensors'
)
"
```

### 方式 3：使用魔搭社区（ModelScope）

```bash
# 1. 安装魔搭库
pip install modelscope

# 2. 下载模型
from modelscope import snapshot_download
model_path = snapshot_download(
    model_id='Qwen/Qwen2.5-7B-4bit',
    cache_dir='Resources/Models'
)
```

## 🔧 编译模型（可选）

### INT4 量化

```python
# 使用 bitsandbytes 量化
from torch import nn
from bitsandbytes importbnb

# 量化模型
model = Qwen25ForCausalLM.from_pretrained("qwen2.5-7b")
model = model.quantize(bnb_4bit=True, bnb_4bit_compute_dtype=torch.float16)

# 导出到 MLX
model.save("model.safetensors")
```
